# Pulsar

Projeto academico de front-end desenvolvido para a Global Solution FIAP 2026/1.

O Pulsar simula uma comunicacao emergencial resiliente entre Defesa Civil, gateways comunitarios e moradores em cenarios de rede instavel. O site apresenta o fluxo de emissao de alerta, propagacao da mensagem, resposta do morador e leitura dos retornos no painel.
