# Pulsar

Projeto acadêmico de front-end desenvolvido para a Global Solution FIAP 2026/1.

O Pulsar simula uma comunicação emergencial resiliente entre Defesa Civil, gateways comunitários e moradores em cenários de rede instável. O site apresenta o fluxo de emissão de alerta, propagação da mensagem, resposta do morador e leitura dos retornos no painel.
